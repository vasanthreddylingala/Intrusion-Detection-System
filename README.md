# Intrusion Detection System for Smart Vehicles

BTech final year project — Bharath Institute of Higher Education and Research, 2025.

## What it does
Detects network intrusions in smart vehicle systems using ML classification models trained on network traffic data.

## Tech stack
Python · scikit-learn · Random Forest · Flask · HTML

## Results
Achieved 92% detection accuracy on network traffic dataset.

## Team
Built as part of a team of 3–4 members.
